RegexBuilder Framework

Comme les regex peuvent parfois et voir même souvent paraître compliquées, Apple nous livre avec Swift 5.7 un framework permettant de simplifier ces opérations.

Le framework RegexBuilder est disponible pour iOS > 16.0+, macOS > 13.0+, watchOS > 9.0+, tvOS > 16.0+, Mac Catalyst 13.0+

Premier Exemple

Ce framework va nous permettre de simplifier l’écriture des regex et de passer de :

let regex1 = /\d+@\w+/

À

let regex2 = Regex {
  OneOrMore(.digit)
  "@"
  OneOrMore(.word)
}

Dans les deux cas l’utilisation restera la même :

import Foundation
import RegexBuilder


let regex1 = /\d+@\w+/
let regex2 = Regex {
    OneOrMore(.digit)
    "@"
    OneOrMore(.word)
}


let match1 = "12345@hello".firstMatch(of: regex1)
let match2 = "12345@hello".firstMatch(of: regex2)


print(match1!.0)
print(match2!.0)

Dans cet exemple \d+ est remplacé par OneOrMore^1[^2] et \w+ par OneOrMore^3 Ce qui rend le tout plus rapidement compréhensible.

Quantificateurs

OneOrMore est un quantificateur signifiant comme son nom l’indique: un ou plus, si on veut rendre une partie optionnelle il suffira d’utiliser Optionnally, dans notre exemple si on veut que la partie avant le @ soit optionnelle il suffira d’écrire : 

let regex3 = Regex {
    Optionally(.digit)
    "@"
    OneOrMore(.word)
}


let match3 = "@hello".firstMatch(of: regex3)
print(match3!.0)

On peut aussi spécifier un nombre précis d’occurrences, par exemple si on souhaite valider une chaine composée de 4 à 6 caractères suivi du symbole @ on écrira :

let regex4 = Regex {
    Repeat(4...6) {
        CharacterClass("a"..."z", "A"..."Z")
    }
    "@"
}


let match4 = "azbrZ@".firstMatch(of: regex4)
print(match4!.0)

Résumé des quantificateurs disponibles :

  • One: Une et une seule occurence

  • ZeroOrMore: 0 à n occurrences

  • Local: Groupe atomique

  • OneOrMore: Un ou plus

  • Repeat: Nombres d’occurrences précisé

Exemple plus complet

Imaginons les deux phrases suivantes: 

let text1 = "Analyste/Dimitri Yann Tavan/$1,000/April 12, 2023"
let text2 = "Illustrateur/Kevin O'Reilly/$400/November 30, 2023"

Dans cet exemple, l’entreprise X doit payer 1000 à Dimitri au plus tard le 12 avril 2023 pour ces services d’analyste et 400 à Kevin au plus tard le 30 novembre 2023, pour son travail en tant qu’illustrateur.

Utilisation de ChoiceOf :

Commençons par la nature de l’emploi. Conformément aux deux textes précédents, il y a deux possibilités : analyste et illustrateur. Donc nous pouvons écrire notre regex comme suit :

let job = Regex {
    ChoiceOf {
        "Analyste"
        "Illustrateur"
    }
}

Maintenant testons sur le premier texte :

if let jobMatch = text1.firstMatch(of: job) {
    let wholeMatch = jobMatch.output
    print(wholeMatch)
}

Ce qui nous sonne en sortie :

Analyste

Comme prévu, donc la première regex est validée.

Utilisation de CharacterClass:

Passons maintenant au composant suivant: le nom. Ce dernier est défini par un ou plusieurs caractères, des espaces optionnels et dans le second texte par un simple caractère ‘

let name = Regex {
    OneOrMore {
        ChoiceOf {
            CharacterClass(.word)
            CharacterClass(.whitespace)
            "'"
        }
    }
}

Si on teste :

if let nameMatch = "Dimitri Yann Tavan".firstMatch(of: name) {
    let wholeMatch = nameMatch.output
    print(wholeMatch)
}

On obtient bien le nom en sortie

Devise (Currency)

Maintenant passons au composant suivant, le paiement. Le texte à trouver est ici $1,000 ou $400. Là encore la manière d’écrire l’expression est grandement simplifiée:

let USLocale = Locale(identifier: "en_US")
let payment = Regex {
    One(.localizedCurrency(code: "USD", locale: USLocale))
}


if let paymentMatch = text1.firstMatch(of: payment) {
    let wholeMatch = paymentMatch.output
    print(wholeMatch)
}

Et en sortie on obtient bien le chiffre 1000.

Maintenant changeons la devise pour l’Euro :

let text3 = "Photographe/David Tavan/999,12 €/9 Mai 2023"
let FRLocale = Locale(identifier: "fr_FR")
let payment2 = Regex {
    One(.localizedCurrency(code: "EUR", locale: FRLocale))
}


if let paymentDavid = text3.firstMatch(of: payment2) {
    let match = paymentDavid.output
    print(match)
}

On obtient bien 999.12 en sortie

Extraction de la date

Tout comme la devise nous allons écrire une expression semblable pour la date. :

let date = Regex {
    One(.date(.long, locale: USLocale, timeZone: .gmt))
}


if let date1 = text1.firstMatch(of: date) {
    let match = date1.output
    print(match)
}

Ce qui nous renvoie bien la date :

2022-04-12 00:00:00 +0000

Capture des éléments

Maintenant nous voulons rassembler toutes les expressions pour parcourir le texte en entier, sachant que chaque partie du texte est séparée par un / Nous écrirons :

let separator = Regex { "/" }


let regexCode = Regex {
    job
    separator
    name
    separator
    payment
    separator
    date
}

Puis pour tester :

if let text1Parsed = text1.firstMatch(of: regexCode) {
    let wholeMatch = text1Parsed.output
    print(wholeMatch)
}
if let text1Parsed = text2.firstMatch(of: regexCode) {
    let wholeMatch = text1Parsed.output
    print(wholeMatch)
}

La sortie est comme prévu :

Analyste/Dimitri Yann Tavan/$1,000/April 12, 2022
Illustrateur/Kevin O'Reilly/$400/November 30, 2022

Maintenant nous allons capturer les éléments en écrivent autour des parties à extraire le mot clé Capture :

let regexCode = Regex {
    Capture {
        job
    }
    separator
    Capture {
        name
    }
    separator
    Capture {
        payment
    }
    separator
    Capture {
        date
    }
}

Ce qui va nous renvoyer un tuple avec les éléments demandés :

if let text1Parsed = text1.firstMatch(of: regexCode) {
    let wholeMatch = text1Parsed.output
    print(wholeMatch.0)
    print("Job: \(wholeMatch.1)")
    print("Nom : \(wholeMatch.2)")
    print("Salaire: \(wholeMatch.3)")
    print("Date de paiement: \(wholeMatch.4)")
}

En sortie :

Analyste/Dimitri Yann Tavan/$1,000/April 12, 2022
Job: Analyste
Nom : Dimitri Yann Tavan
Salaire: 1000
Date de paiement: 2022-04-12 00:00:00 +0000

Et avec un peu de formatage :

let dateFormatter = DateFormatter()
dateFormatter.locale = Locale(identifier: "fr_FR")
dateFormatter.dateStyle = .medium


let currencyFormatter = NumberFormatter()
currencyFormatter.locale = FRLocale
currencyFormatter.numberStyle = .currency
currencyFormatter.currencyCode = "EUR"


if let text1Parsed = text1.firstMatch(of: regexCode) {
    let (all, job, name, salary, date) = text1Parsed.output
    print(all)
    print("Job: \(job)")
    print("Nom : \(name)")
    print("Salaire: \(currencyFormatter.string(from: salary as NSNumber)!)")
    print("Date de paiement: \(dateFormatter.string(from: date))")
}

On obtient (je sais ça fausse tout mais c’est plus zoli 😜 :

Analyste/Dimitri Yann Tavan/$1,000/April 12, 2022
Job: Analyste
Nom : Dimitri Yann Tavan
Salaire: 1 000,00 €
Date de paiement: 12 avr. 2022

Transformation des valeurs capturées

Il est aussi possible de transformer au sein même de l’expression les valeurs capturées, dans l’exemple suivant, on extrait les nombres, puis on les transforme avant de les ajouter

let message = "12 unités, en stocks 120"


let regexStock = Regex {
    Capture {
        OneOrMore(.digit)
    } transform: { value in
        Int(value)
    }
    
    OneOrMore(.any, .reluctant)
    
    Capture {
        OneOrMore(.digit)
    } transform: { value in
        Int(value)
    }
}


let result = message.matches(of: regexStock)


if let units = result.first?.output.1,
   let stock = result.first?.output.2 {
    let total = units + stock
    print("Stock Final : \(total)")
}

Ce qui nous donne bien en sortie : 

Stock Final : 132

Autre exemple de capture

Soit le texte suivant : 

let greetings2 = "Mon nom est David et j'ai 53 ans."

Pour extraire le nom et l’âge :

let search2 = Regex {
    "Mon nom est "
    
    Capture {
        OneOrMore(.word)
    }
    
    " et j'ai "
    
    TryCapture {
        OneOrMore(.digit)
    } transform: { match in
        Int(match)
    }
    
    " ans."
}


if let resultSearch = greetings2.firstMatch(of: search2) {
    let(_, nom, age) = resultSearch.output
    print("Nom : \(nom)")
    print("Age: \(age)")
}

Trouver tous les mots commençants par une des deux lettres (Di)

let testText = "Coucou comment ça va aujourd'hui Dimitri ? Il fait 30 dehors"


let regexDA = Regex {
    Anchor.wordBoundary
    One(.anyOf("D"))
    ZeroOrMore {
        .word
    }
}.ignoresCase()


let matches = testText.matches(of: regexDA)


matches.forEach { print(testText[$0.range.lowerBound..<$0.range.upperBound]) }

Pour trouver les mots commençants par une lettre précise on écrira :

let regexDA = Regex {
    Anchor.wordBoundary
    One("D")
    ZeroOrMore {
        .word
    }
}.ignoresCase()
